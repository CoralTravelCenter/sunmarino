Почему апрель?

Моковая секция для теста sticky-nav. Здесь можно прокруткой проверить, как компонент переключает активное состояние по IntersectionObserver.

Турция

Секция с отдельным id. Навигационная ссылка должна становиться активной, когда линия активации sticky-nav попадает в эту область.

Египет

Высота сделана намеренно большой, чтобы было удобно тестировать переходы активного пункта при скролле вниз и вверх.

Мальдивы

Эта секция нужна только для локального playground и не влияет на публичный API библиотеки.

Таиланд

Последняя тестовая секция. По ней удобно проверить финальное active-состояние ближе к нижней части страницы.
