Почему апрель? Турция Египет Мальдивы Таиланд

Почему апрель?

Моковая секция для теста sticky-nav. Здесь можно прокруткой проверить, как компонент переключает активное состояние по IntersectionObserver.

Турция

Секция с отдельным id. Навигационная ссылка должна становиться активной, когда линия активации sticky-nav попадает в эту область.

Египет

Высота сделана намеренно большой, чтобы было удобно тестировать переходы активного пункта при скролле вниз и вверх.

Мальдивы

Эта секция нужна только для локального playground и не влияет на публичный API библиотеки.

Таиланд

Последняя тестовая секция. По ней удобно проверить финальное active-состояние ближе к нижней части страницы.

Почему апрель? Турция Египет

Почему апрель?

Тестовый контент панели для проверки базовой логики tabs в playground. Табы можно найти здесь https://www.sunmar.ru/familyholiday/

Турция

Эта панель выбрана по умолчанию через value на sunmar-tabs.

Египет

Третья панель нужна для ручной проверки переключения и keyboard navigation.

Accordion — multiple

Какие документы нужны?

Это тестовый контент для проверки базового сценария multiple. Здесь можно проверить открытие и закрытие item независимо друг от друга.

Можно ли перенести тур?

Вторая секция нужна для проверки обычного переключения и визуального контракта trigger/content.

Что входит в стоимость?

Третий item нужен для ручной проверки отступов, иконки и общего поведения family в playground.

Accordion — single

Почему открыт только один item?

В режиме single контейнер сам нормализует состояние и не позволяет держать открытыми несколько пунктов одновременно.

Что происходит при повторном клике?

Этот пункт нужен для проверки, что в single-режиме активный item можно закрыть повторным кликом по trigger.

Disabled item

Disabled-сценарий нужен для ручной проверки визуального состояния и блокировки взаимодействия.
